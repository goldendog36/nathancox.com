## Local Development

To run locally:
htt
1. Start a simple HTTP server:

   ```bash
   python3 -m http.server 8000
   ```

2. Open http://localhost:8000 in your browser.

## Deployment

To deploy to GitHub Pages:

```bash
npm run deploy
```

This will publish the current directory to the `gh-pages` branch.
